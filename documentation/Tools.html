Module Tools


module Tools: sig .. end
Implements several frequently used functions.
type pos = string * int * int
Type to describe a point in a source file.
val ln : 'a * 'b * 'c -> 'b
Returns the second element of a tuple of 3 elements. For values of type Tools.pos, it correspond to the line number in source file.
val cn : 'a * 'b * 'c -> 'c
Returns the third element of a tuple of 3 elements. For values of type Tools.pos, it correspond to the column number in source file.
val fn : 'a * 'b * 'c -> 'a
Returns the first element of a tuple of 3 elments. For values of type Tools.pos, it correspond to the source-file name.
val no_pos : string * int * int
Defines a null Tools.pos.
val string_of_pos : string * int * int -> string
Returns a string representation of a Tools.pos value.
val string_of_set : ('a -> 'b) ->
       (('a -> 'b list -> 'b list) -> 'c -> 'd list -> string list) -> 'c -> string
Prints the elements of a set, separated by commas.
f : Function used to build a string representation of each element in the set.
fold : Function used to reduce the set.
set : A set.
val string_of_map : ?swap:bool ->
       ('a -> string) ->
       ('b -> string) ->
       (('a -> 'b -> string list -> string list) -> 'c -> 'd list -> string list) ->
       'c -> string
Prints the entries of a map, separated by commas.
swap : If true, swap keys and values of the map. Optional, default is false.
f1 : Function used to build a string representation of each entry's key.
f2 : Function used to build a string representation of each entry's value.
map : A map.
val string_of_array : ('a -> string) -> 'a array -> string
Returns a string representation of an array.
f : Function used to get a string representation of each element in the array.
ar : An array.
val string_of_list : ('a -> string) -> 'a list -> string
Returns a string representation of a list.
f : Function used to get a string representation of each element in the list.
l : A list.
val pow : int -> int -> int
val bit_rep_size : int -> int
Returns the number of bits needed to represent n in base 2.
n : An integer.
val replace_space : string -> string
Replace spaces by underscores in a given string and returns it.
str : A string.
val read_input : unit -> string
Reads standard input and returns it without spaces.
